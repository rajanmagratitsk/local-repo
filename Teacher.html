Java Basics

Java is a high-level, class-based, object-oriented programming language.

Key points:

  • Platform independent (Write Once Run Anywhere)
  • Runs on JVM
  • Strongly typed language

Java data types:

  • int, float, double, char, boolean
  • String (class)
int age = 25; double price = 9.99; char grade = 'A'; boolean isActive = true;

Conditionals:

if(age > 18){ System.out.println("Adult"); }else{ System.out.println("Minor"); }

Loops:

for(int i=0; i<5; i++){ System.out.println(i); } while(true){ break; }

Object-Oriented Programming

class Student { String name; int age; void display(){ System.out.println(name + " " + age); } } public class Main { public static void main(String[] args){ Student s1 = new Student(); s1.name = "Rajan"; s1.age = 21; s1.display(); } }
class Person { String name; } class Teacher extends Person { String subject; }

Polymorphism: Method overloading & overriding

Encapsulation: Using private variables and getter/setter methods
