Features of HTML

Hyper Text Markup Language

1. Simple and Easy

HTML is very easy to learn and use. It uses simple tags to create web pages.

2. Platform Independent

HTML pages can run on any operating system like Windows, Linux, and Mac.

3. Supports Multimedia

HTML allows adding images, audio, video, and animations in web pages.

4. Hyperlink Support

HTML provides links to connect one page to another page using anchor tags.

5. Used with CSS and JavaScript

HTML works with CSS for design and JavaScript for interactivity.

6. Free and Open Standard

HTML is free to use and supported by all major web browsers.
