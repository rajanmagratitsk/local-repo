Statistics and Probability Formulas

1. Measures of Central Tendency

Mean = ΣX / N
Median = (N+1)/2 th item
Mode = Most Frequent Value

2. Measures of Dispersion

Range = Largest Value - Smallest Value
Variance = Σ(X - Mean)² / N
Standard Deviation = √Variance
Coefficient of Variation = (SD / Mean) × 100

3. Probability Formulas

P(A) = Number of favorable outcomes / Total outcomes
0 ≤ P(A) ≤ 1
P(A') = 1 - P(A)
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
P(A ∩ B) = P(A) × P(B) (Independent Events)

4. Permutation and Combination

nPr = n! / (n-r)!
nCr = n! / r!(n-r)!
n! = n × (n-1) × (n-2)...1

5. Correlation & Regression

Karl Pearson Correlation:
r = [NΣXY - (ΣX)(ΣY)] / √[(NΣX²-(ΣX)²)(NΣY²-(ΣY)²)]

Regression Line:
Y = a + bX

6. Binomial Distribution

P(X=x) = nCx p^x q^(n-x)
Mean = np
Variance = npq

7. Normal Distribution

Z = (X - μ) / σ

8. Hypothesis Testing

t = (Sample Mean - Population Mean) / (S / √n)
χ² = Σ(O - E)² / E
F = Variance₁ / Variance₂

9. Index Number

Laspeyres Index = (ΣP₁Q₀ / ΣP₀Q₀) × 100
Paasche Index = (ΣP₁Q₁ / ΣP₀Q₁) × 100

10. Time Series

Trend Line: Y = a + bX
